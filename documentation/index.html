Install Instructions

Requirements: Apache 2.2+, Mysql 5.4+ (only_full_group_by should be disabled), PHP 5.6+
 
  1. Copy/Upload the folder and files to your web server using cPanel or FTP
  2. If you are installing in sub folder, please make sure there is no space in folder names
  3. Create mysql database, database user and grant all permission to user for this database.
  4. Open the http://yourdomain.com/pathtoupload/install/index.php
    1. Step 1: Pre-install system checklist - If there is any issue with checklist, you will need to fix it first to proceed to next step.
    2. Step 2: Provide you Envato Username and Purchase Code to validate the purchase.
    3. Step 3: Provide the database host, database user, database password and database name to proceed to next step. Please make sure you don't have $ sign in your password.
    4. Step 4: Site Config: Please Fill in you RMS ADVANCE Base URL and select your timezone. You must check the site url as installer might guess it wrong
      • Base URL for public_html or www/htdocs folder will be http://yourdomain.com/ or
      • Base URL for sub-domain will be 
      • Base URL for sub-folder will be
      • Base URL must not have space and it should end with a trailing slash
  5. If everything goes smooth, you will be redirect to final page with login details.
  6. Remove the install folder before start using the RMS ADVANCE.
  7. Login using default username and password, and edit user to change login details.
  8. RMS ADVANCE is ready to use. Please save system settings before start using it.

 

 

After Install

  1. Add new owner user before editing/deleting the default owner account (we have disabled the option to edit the logged in user email and user name, you won't be able edit/delete the default user unless you are logged in with new owner account).
  2. Update the system settings, by selecting the default currency and all other necessary things.
  3. You are ready to go, start adding repairs, users, customer purchases, vendor purchases, accounts, expenses, deposits, taxes, stores, products, customer, sales and purchase and much more.

Users

Users are the staff members (your employees). You can add any number of users and assign them group (manager groups and permissions in settings).
Admin user groups are with fixed permissions and any other user group added by you, can have custom permissions.
Please check the edit user page for some user related permission such as view/edit rights.

Customers (small CRM)

Customers are the clients of your companies/organizations. They purchase the items that you/your companies/organizations sell.
Company, Name (Contact Person), Email, Phone, Address and City input fields are required. you can edit mandatory fields in settings. and also, add multiple data for customers ie. Activities, Internal Notes. view all the purchases, refunds, repairs history
The inputs for customer in other modules are no more select/drop down but ajax calls, you can type in search filed to get the suggestions and select any suggestion available in database for customers. You can add customer by clicking the Add Customer from any page. The interface is quite easy to understand.

Reports

Reports

The following reports are available

  1. Stock Report: The chart for stock value
  2. Finance Report: The record for finance
  3. Sales Report:   The record for sales  
  4. Returned Sales: The record for Returned sales
  5. Profit Report: Profit and/or Loss report for the selected time frame (date range).
  6. Tax Report:    The record for each tax on all sales
  7. Vendor Purchases:   The record for Vendor Purchases  
  8. Customer Purchases:  The record for Customer Purchases
  9. Drawer Report: All record of register/counters for POS.
  10. General Ledger: General Ledger Report
  11. Activities: Customer Activities 
  12. Commission Report: Employee Commisions record on each item. 
  13. Quantity Alerts: The items that have reached the low stock (alert quantity).
  14. Counted Stock: The record for items that have been counted.
